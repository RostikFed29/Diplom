Інструкція з запуску програми через PyCharm
✅ Крок 1: Встановіть PyCharm
Завантажте PyCharm з офіційного сайту: https://www.jetbrains.com/pycharm/download

Встановіть та відкрийте PyCharm.

✅ Крок 2: Створіть новий проєкт
У вікні PyCharm натисніть "New Project".

Виберіть місце збереження проєкту та натисніть "Create".

✅ Крок 3: Додайте програмний файл
У проєкті клацніть правою кнопкою на папку project_name, виберіть "New" → "Python File".

Назвіть файл, наприклад: image_restore_app.

Скопіюйте весь код програми (який ви надали) у цей файл.

✅ Крок 4: Встановіть необхідні бібліотеки
Відкрийте термінал у PyCharm (нижня панель).

Введіть ці команди по черзі:

pip install Pillow
pip install opencv-python

✅ Крок 5: Запуск програми

Нажміть кнопку запуску програми у pycharm або зажміть клавіші shift+f10

Результат: відкриється вікно графічного інтерфейсу програми для відновлення зображень.
